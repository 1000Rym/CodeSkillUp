## DataType
### Difference between floating and double.
|           Float           |	        Double          |
|           :---:           |           :---:           |
| Single precision          | Double precision          |
| 7 significant digits      | 15 significant digits     |
| 4 bytes (32 bits IEEE 754)| 8 bytes (64-bits IEEE 754)|
| Cost-Effective            | Precision                 |
